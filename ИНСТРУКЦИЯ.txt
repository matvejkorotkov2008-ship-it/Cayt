ИНСТРУКЦИЯ ПО НАСТРОЙКЕ САЙТА BMJAN
=====================================

Сайт автоматически пытается загрузить посты и фотографии из Telegram канала @BMJAN.

ТЕКУЩАЯ РАБОТА:
---------------
Сайт использует публичные API для получения данных из канала. 
Откройте index.html в браузере - он должен автоматически загрузить посты и фотографии.

АВТОМАТИЧЕСКОЕ ОБНОВЛЕНИЕ:
--------------------------
Сайт автоматически обновляет посты и фотографии каждые 5 минут.

ЕСЛИ ДАННЫЕ НЕ ЗАГРУЖАЮТСЯ:
----------------------------
1. Убедитесь, что канал @BMJAN публичный
2. Проверьте интернет-соединение
3. Откройте консоль браузера (F12) для просмотра ошибок

АЛЬТЕРНАТИВНЫЙ МЕТОД (для более надежной работы):
-------------------------------------------------
Если публичные API не работают, можно использовать Telegram Bot API:

1. Создайте бота через @BotFather в Telegram
2. Добавьте бота как администратора в канал @BMJAN
3. Откройте файл config.js
4. Вставьте токен бота в поле BOT_TOKEN
5. Сохраните файл

СТРУКТУРА ФАЙЛОВ:
-----------------
- index.html - главная страница
- styles.css - стили сайта
- script.js - логика загрузки данных
- config.js - настройки (опционально)

ОТКРЫТИЕ САЙТА:
---------------
Просто откройте файл index.html двойным кликом в браузере.
